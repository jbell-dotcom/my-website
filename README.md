# Paramount Resources Director Dashboard Prototype

A polished static prototype for a web-based director dashboard for Paramount Resources Ltd. (TSX: POU).

## What's included

- POU-branded visual treatment using an earth-tone energy palette inspired by Paramount's public website.
- Section 1: Market Data with peer quote table and commodity/FX API slots.
- Section 2: Top Stories with category filters.
- Section 3: Market Views for analyst themes and board-prep questions.
- Section 4: Public Documents with direct investor-page links.

## Production data notes

Realtime market data should be connected to a licensed data source such as TMX, Bloomberg, FactSet, Refinitiv/LSEG, S&P Capital IQ, or another approved provider. Broker research should only be summarized through licensed access and should not be scraped from restricted sources.

Open `index.html` in a browser to view the prototype.
